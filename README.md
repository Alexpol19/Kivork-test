# Voyage - KIVORK test project bootstrapped on Pug, Sass, Bootstrap4

#### You can see demo here [demo](https://github.com/Alexpol19/Kivork-test/blob/main/build/index.html)

## How to install
### In the project directory you can run
```bash 
npm install
```
## How to start 
### In the project directory you can run
```bash 
npm start
```
